# 🤖 בוט ווטסאפ → גוגל דרייב
### לשכת ראש העיר צפת

---

## מה הבוט עושה?
כל תמונה או סרטון שנשלחים בקבוצה הווטסאפ → עולים אוטומטית לתיקיה בגוגל דרייב, ממוינים לפי תאריך.

**מבנה הקבצים בדרייב:**
```
📁 לשכת ראש העיר (תיקיית הבסיס)
  📁 26-2-2026
    📄 14-30-00_יוסי_כהן.jpg
    📄 14-35-22_דינה_לוי.mp4
  📁 27-2-2026
    📄 ...
```

---

## שלב 1 — התקנת Node.js
הורד והתקן מ: https://nodejs.org (גרסה 18 ומעלה)

---

## שלב 2 — הגדרת גוגל דרייב API

1. כנס ל: https://console.cloud.google.com
2. צור **פרויקט חדש** (שם: `whatsapp-bot`)
3. חפש **"Google Drive API"** → לחץ **הפעל**
4. לחץ **"יצירת אישורים"** → בחר **"OAuth 2.0 Client ID"**
5. סוג אפליקציה: **"Desktop App"**
6. לחץ **הורד JSON** → שמור כ-`credentials.json` בתיקיית הפרויקט

---

## שלב 3 — מצא את ID התיקיה בדרייב

1. פתח את גוגל דרייב
2. צור תיקיה חדשה: **"לשכת ראש העיר - תמונות"**
3. לחץ עליה, הכתובת תיראה כך:
   `https://drive.google.com/drive/folders/`**`1ABC123xyz...`**
4. העתק את החלק המודגש — זה ה-`DRIVE_FOLDER_ID`

---

## שלב 4 — התקן את הבוט

פתח Command Prompt / Terminal בתיקיית הפרויקט:

```bash
npm install
```

---

## שלב 5 — הגדר את שם הקבוצה

ערוך את קובץ `.env` (צור אותו אם לא קיים):

```
DRIVE_FOLDER_ID=הדבק_כאן_את_ה_ID_שהעתקת
GROUP_NAME=לשכת ראש העיר
```

---

## שלב 6 — אמת מול גוגל (פעם אחת בלבד!)

```bash
node auth.js
```

דפדפן ייפתח → התחבר לחשבון גוגל → אשר הרשאות → חזור לטרמינל.

---

## שלב 7 — הפעל את הבוט

```bash
node index.js
```

יופיע **QR Code** בטרמינל:
1. פתח ווטסאפ בנייד
2. הגדרות → מכשירים מקושרים → **קשר מכשיר**
3. סרוק את הקוד

הבוט פעיל! 🎉

---

## הפעלה אוטומטית (אופציונלי)

כדי שהבוט יפעל תמיד גם לאחר הפעלה מחדש של המחשב, הרץ:

```bash
npm install -g pm2
pm2 start index.js --name whatsapp-bot
pm2 startup
pm2 save
```

---

## פתרון בעיות

| בעיה | פתרון |
|------|--------|
| הקבוצה לא נמצאת | שנה `GROUP_NAME` ב-.env לשם המדויק |
| שגיאת אימות גוגל | הרץ שוב `node auth.js` |
| הבוט התנתק | הרץ שוב `node index.js` |
| קבצים לא עולים | ודא שה-`DRIVE_FOLDER_ID` נכון |

---

## נדרש: Node.js 18+, חשבון גוגל עם Google Drive
